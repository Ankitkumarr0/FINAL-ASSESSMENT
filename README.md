# SmartCampus Management System

## 📌 Problem Statement

A Java-based system to manage students, courses, and enrollments efficiently with multithreading and exception handling.

## 🚀 Features

* Add Students
* Add Courses
* Enroll Students
* View Data
* Multithreading for processing
* Custom Exception Handling

## 🛠️ Technologies Used

* Java
* OOP Concepts
* Collections (HashMap, ArrayList)
* Multithreading

## ⚙️ How to Run

1. Compile:
   javac SmartCampus.java
2. Run:
   java SmartCampus

## 📸 Output

(Menu-driven console program showing enrollment system)

## ✨ Unique Feature

* Thread-based enrollment simulation
